1.Як можна оголосити змінну в JavaScript?
Змінну можна оголосити за допомогою:

  • var
  • const
  • let

2.У чому різниця між функцією prompt та функцією confirm?

Функція prompt може приймати два аргументи. І вона повертає значення надруковане в полі вводу текст або null. Функція confirm показує повідомлення і чекає, поки користувач натисне ОК або Відміна. Повертає true або false.

3.Що таке неявне перетворення типів?

Неявне перетворення – коли інтерпретатор автоматично виконує перетворення типів, т. е. без участі програміста.
